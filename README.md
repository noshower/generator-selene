# 安装

### 方式 1（推荐）：

**注意**：如果之前全局安装了 yo 或 generator-selene, 请先确保已经卸载掉！

```js
// 新建项目目录
mkdir my-new-project && cd my-new-project

// 运行命令
npm init yo selene
```

### 方式 2：

```js
// 全局安装
npm install -g yo generator-selene

// 新建项目目录
mkdir my-new-project && cd my-new-project

// 运行命令
yo selene
```

# 启动应用

```js
npm start
```
